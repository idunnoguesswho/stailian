STAILIAN

An elemental world of exploration, strategy, and crafting

Stailian now opens through static HTML on GitHub Pages, with the active game screens backed by a shared Google Sheet in the browser. Roll across the map, build your wardrobe, and track your scorecard without a PHP server.
